**Brief: Corporate Grid Material Design Dashboard**

## 1. Design Mandate

Transform Leon's "Autonomous Coder" UI into a **corporate-grade, space-efficient Material Design dashboard**.

### Source Reference
- Original code: `_reference/autocoder/ui/` (React/Vite)
- Original look: Functional but amateur (poor space utilization, garish colors)

### Target Aesthetic
- **Corporate Grid**: Clean, professional, information-dense
- **Material Design 3**: MUI v5 components
- **Tailwind**: Precise positioning, utility classes
- **Color Palette**: Professional blues/grays, subtle accents
- **Typography**: Inter or Roboto, clear hierarchy
